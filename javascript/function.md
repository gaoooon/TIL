# function

## function 기본 구조

```js
function [함수 이름]() {
    실행할 코드
}
// 실행할때

[함수이름]();

```

## function에 매개변수를 넣을때

```js
function [함수 이름](매개변수1, 매개변수2) {
    실행할 코드
}
// 괄호안에 넣고 쉼표로 구분한다
```

#### example

```js
function hello(name) {
  console.log("hello my name is", name);
}

hello("방가온");

// "hello my name is 방가온" 이런식으로 나온다
```
